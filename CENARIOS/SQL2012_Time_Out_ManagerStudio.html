AUMENTO DO TIMEOUT NO MANAGEMENT STUDIO.

Foi aberto um chamado solicitado para equipe de suporte informando que o SQL Server Manager Studio

não estava conseguindo conectar na base de dados dos servidores da equipe de desenvolvimento.

									1. Execute o programa SQL Server Management Studio no terminal do usuário;

										

									2. Em seguida, clique em OPTIONS conforme a imagem no exemplo abaixo;;

										

									3. Na tela seguinte será exibida a guia CONNECTIONS PROPERTIES,
									onde por padrão o tempo de conexão é de 15 segundos;

										

									4. Altere para 30 segundos igual a imagem no exemplo abaixo;

										

									5. Por ultimo, clique em LOGIN para realizar a conexão no banco de dados;


									Procedimento realizado no dia 04/10/2018.
