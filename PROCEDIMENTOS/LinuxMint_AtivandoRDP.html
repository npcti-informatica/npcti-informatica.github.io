ATIVANDO REMOTE DESKTOP NO LINUX MINT.


								Fonte: http://www.informaticanaweb.com/remote-desktop-ubuntu/

								Procedimentos para ativar o serviço RDP em uma máquina que está com Linux MINT.	

								1. Abra o terminal do Linux MINT;

								

								2. Inicialmente execute o comando IFCONFIG para ver o endereço IP que foi atribuído a máquina;

								

								3. Após o procedimento, execute o comando sudo apt-get install xrdp para instalar o pacote XRDP no Linux MINT;

								Logo após apertar ENTER o sistema vai solicitar a senha do LOGIN máster para continuar com a instalação;

								

								4. Após concluir, realize um LOGOFF na máquina;

								

								5. Em uma máquina Windows, execute a ferramenta REMOTE DESKTOP e informe o IP do micro cliente LINUX que foi habilitado o acesso remoto;

								

								6. Será exibida uma mensagem informando que ocorreu um problema ao executar o REMOTE DESKTOP, aperte YES para prosseguir;

								

								7. Após confirmar, será exibida uma nova tela de autenticação;

								

								8. O acesso remoto a máquina foi realizada com sucesso;
