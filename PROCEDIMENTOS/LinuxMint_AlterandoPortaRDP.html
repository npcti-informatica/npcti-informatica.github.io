LINUX MINT – ALTERANDO O NUMERO DA PORTA RDP.


								Fonte: https://www.blog.flaviodeoliveira.com.br/2014/08/instalando-o-rdp-remote-desktop.html

								Procedimentos para alterar o número da porta RDP no Linux MINT que por padrão 3389.	

								1. Após realizar o LOGON no Linux MINT abra o console do terminal COMMAND PROMPT;

									

								2. Caso não tenha instalado o pacote XRDP para realizar o acesso remoto , execute o comando sudo apt-get install xrdp;

								3. Após o procedimento, execute o comando sudo nano /etc/xrdp/xrdp.ini para abrir o conteúdo do arquivo de configuração do XRDP;

								

								4. Por padrão a porta configurada é 3389 conforme o exemplo na imagem abaixo;

								

								5. Altere o número da porta. Para este laboratório a porta será alterada para 3004;

								

								6. Após a alteração pressione as teclas CTRL + X para sair, onde o sistema perguntara se deseja salvar as alterações;

								

								7. O sistema vai perguntar se deseja salvar as informações. Digite Y e aperte ENTER;

								

								8. Retornando para tela principal, execute o comando sudo service xrdp restart para reiniciar o serviço XRDP;

								

								9. Realize o LOGOFF no sistema;

								

								10. No Windows execute a ferramenta para acesso RDP;

								

								11. Em seguida, será exibida a tela de LOGIN da ferramenta XRDP do LINUX;

								

								12. Após autenticar, o desktop da conta do usuário será exibida com sucesso;
